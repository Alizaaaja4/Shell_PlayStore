## About 📖

PlayStore is a hands-on project designed to enhance your proficiency in shell scripting (Bash) by building a straightforward interface. The primary objective is to offer practical exercises in Bash scripting, focusing on developing simple yet effective applications. Through this project, you'll learn how to manage data, manipulate strings, gather user input, and execute system commands using shell scripts.
